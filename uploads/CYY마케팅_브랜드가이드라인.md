## 1. 브랜드 아이덴티티

### 브랜드 핵심 키워드

```
신뢰 · 성장 · 실용 · 파트너십
```

### 브랜드 포지셔닝

CYY마케팅은 **"중소기업과 소상공인 곁에 있는 든든한 마케팅 파트너"** 입니다.  
화려한 기술보다 **실질적인 성과**, 복잡한 전략보다 **명확한 실행**을 우선합니다.

### 브랜드 퍼소나

| 항목 | 내용 |
|------|------|
| **성격** | 친근하고 전문적인, 믿음직한 사수 |
| **말투** | 쉽고 명확하게, 전문 용어는 풀어서 설명 |
| **태도** | 고객의 입장에서 먼저 생각하는 |
| **이미지** | 깔끔하고 모던하지만 딱딱하지 않은 |

### 브랜드 약속 (Brand Promise)

> "복잡한 마케팅, CYY가 쉽게 만들어드립니다."

## 2. 로고 사용 규정

### 로고 기본형

- **국문 로고:** `CYY마케팅`
- **영문 로고:** `CYY Marketing`
- **심볼 + 로고 조합형:** 심볼마크와 브랜드명을 함께 사용하는 기본 형태

### 최소 사용 크기

| 매체 | 최소 너비 |
|------|----------|
| 인쇄물 | 25mm |
| 디지털 (웹/앱) | 80px |
| 명함 | 20mm |

### 여백 규정 (Clear Space)

로고 주변에는 로고 높이의 **1/2 이상**의 여백을 확보해야 합니다.  
다른 요소(텍스트, 이미지, 아이콘)가 이 영역을 침범해서는 안 됩니다.

### 로고 사용 가능 배경

| 배경색 | 사용 로고 |
|--------|----------|
| 흰색 (#FFFFFF) | 컬러 로고 (기본) |
| 브랜드 블루 (#1A6BFF) | 화이트 로고 |
| 다크 (#1A1A2E) | 화이트 로고 |
| 밝은 그레이 (#F5F6FA) | 컬러 로고 |

## 3. 컬러 시스템

### 주요 컬러 (Primary Colors)

#### 🔵 브랜드 블루 — 메인 컬러
```
HEX:  #1A6BFF
RGB:  26, 107, 255
CMYK: 90, 58, 0, 0
용도: 로고, 주요 버튼, 강조 텍스트, 포인트 요소
```

#### ⬛ 브랜드 다크 — 서브 컬러
```
HEX:  #1A1A2E
RGB:  26, 26, 46
CMYK: 74, 74, 0, 82
용도: 배경, 헤드라인, 고급스러운 인쇄물
```

---

### 보조 컬러 (Secondary Colors)

#### 🟢 성공 그린 — 성과 강조용
```
HEX:  #00C896
RGB:  0, 200, 150
용도: 성과 지표, 긍정 메시지, 인포그래픽 포인트
```

#### 🟡 에너지 옐로 — 강조 포인트용
```
HEX:  #FFB800
RGB:  255, 184, 0
용도: 배지, 태그, 프로모션 강조 요소
```

## 4. 타이포그래피

### 국문 서체

| 용도 | 서체 | 굵기 |
|------|------|------|
| 헤드라인 | **Pretendard** | Bold (700) |
| 서브타이틀 | **Pretendard** | SemiBold (600) |
| 본문 | **Pretendard** | Regular (400) |
| 캡션/주석 | **Pretendard** | Light (300) |

> 📌 Pretendard 미설치 환경에서는 **Apple SD Gothic Neo** 또는 **Noto Sans KR** 대체 사용

### 영문 서체

| 용도 | 서체 | 굵기 |
|------|------|------|
| 헤드라인 | **Inter** | Bold (700) |
| 본문 | **Inter** | Regular (400) |

### 폰트 사이즈 스케일

| 단계 | 크기 | 용도 |
|------|------|------|
| Display | 48px / 36pt | 메인 슬로건, 배너 헤드라인 |
| H1 | 32px / 24pt | 페이지 제목 |
| H2 | 24px / 18pt | 섹션 제목 |
| H3 | 20px / 15pt | 소제목 |
| Body | 16px / 12pt | 일반 본문 |
| Caption | 13px / 10pt | 주석, 출처, 태그 |

### 줄간격 (Line Height)

- 헤드라인: `1.3`
- 본문: `1.7`
- 캡션: `1.5`

## 5. 브랜드 보이스 & 톤

### 브랜드 보이스 원칙

CYY마케팅은 마케팅을 어렵게 느끼는 사람들에게 **쉽고 믿음직한 목소리**로 말합니다.

| 우리는 이렇게 말합니다 | 이렇게는 말하지 않습니다 |
|----------------------|----------------------|
| 쉽고 명확하게 | 전문 용어 남발 |
| 솔직하고 투명하게 | 과장된 수치, 허위 약속 |
| 고객 관점에서 | 회사 중심적 표현 |
| 따뜻하고 친근하게 | 딱딱한 기업 문어체 |
| 자신감 있게 | 불필요한 겸손 표현 |

---

### 상황별 톤 가이드

#### 📢 광고 / 프로모션 카피
- 톤: **에너지 있고 직관적으로**
- 특징: 짧고 강렬한 문장, 행동을 유도하는 CTA
- 예시:
  - ✅ "지금 바로 시작하세요. 첫 달 무료 상담."
  - ❌ "저희 서비스를 한번 경험해 보시겠어요?"

#### 💬 SNS 피드 / 콘텐츠
- 톤: **친근하고 유익하게**
- 특징: 구어체 허용, 이모지 적극 활용, 질문형 문장으로 참여 유도
- 예시:
  - ✅ "인스타 광고, 어디서부터 시작해야 할지 막막하셨죠? 🤔 오늘부터 따라해 보세요!"
  - ❌ "당사의 인스타그램 광고 운영 서비스를 소개합니다."

#### 📧 이메일 / 공식 문서
- 톤: **전문적이면서 따뜻하게**
- 특징: 격식체 유지, 하지만 딱딱하지 않게, 고객 이름 호칭 사용
- 예시:
  - ✅ "안녕하세요, [고객명] 대표님. 이번 달 성과를 함께 살펴보겠습니다."
  - ❌ "수신자 귀중. 당월 마케팅 결과를 아래와 같이 보고드립니다."

#### 📞 상담 / 고객 응대
- 톤: **공감하고 해결 중심으로**
- 특징: 고객의 문제를 먼저 인정, 해결책 제시, 긍정적 마무리
- 예시:
  - ✅ "그 부분이 걱정되셨군요. 저희가 이런 방식으로 해결해 드릴 수 있어요."
  - ❌ "그건 어려운 부분이긴 한데요…"

---

### 자주 사용하는 표현 / 금지 표현

**✅ 권장 표현**
- "함께 성장해요"
- "바로 실행 가능한"
- "투명하게 보여드립니다"
- "맞춤 전략"
- "성과로 증명합니다"

**❌ 금지 표현**
- "업계 최고" (증명 불가한 과장)
- "100% 보장" (허위 약속)
- "타사 대비 압도적" (비교 광고 주의)
- "~하여 드림" (지나치게 낮춤)
- 과도한 영어 혼용 ("ROI maximize하여 ROAS를 boost하세요")

## 6. 콘텐츠 원칙

### 콘텐츠 3원칙

```
1. 유익함 (Useful)    — 읽으면 실제로 도움이 되는가?
2. 진실됨 (Honest)    — 과장 없이 사실에 근거하는가?
3. 행동 유발 (Actionable) — 다음 단계를 명확히 안내하는가?
```

### 콘텐츠 유형별 비율 (SNS 기준)

| 유형 | 비율 | 예시 |
|------|------|------|
| 정보성 콘텐츠 | 50% | 마케팅 팁, 업계 트렌드, 사례 분석 |
| 브랜드 콘텐츠 | 20% | 서비스 소개, 팀 소개, 회사 문화 |
| 고객 성공 사례 | 20% | 후기, 비포&애프터, 수치 공유 |
| 프로모션 | 10% | 할인, 이벤트, 신규 서비스 안내 |

### 해시태그 운영 기준

**기본 해시태그 (항상 포함)**
```
#CYY마케팅 #중소기업마케팅 #소상공인마케팅 #마케팅대행사
```

**콘텐츠 유형별 추가 해시태그**
```
광고 관련:   #네이버광고 #인스타광고 #디지털광고 #온라인광고
SNS 관련:   #SNS마케팅 #인스타마케팅 #콘텐츠마케팅
소상공인:   #소상공인 #창업 #자영업 #스타트업마케팅
```

> 📌 게시물당 해시태그는 **최대 15개** 이내로 유지

## 7. SNS 채널별 운영 기준

### 인스타그램

| 항목 | 기준 |
|------|------|
| 게시 빈도 | 피드 주 3회, 스토리 매일 1~2회 |
| 이미지 비율 | 피드 1:1 또는 4:5, 스토리 9:16 |
| 캡션 길이 | 150자 이내 (핵심 먼저, 상세 내용은 댓글) |
| 톤 | 친근하고 짧게, 이모지 2~3개 |
| CTA | "프로필 링크 클릭 👆" 또는 "DM 주세요!" |

### 블로그 (네이버)

| 항목 | 기준 |
|------|------|
| 게시 빈도 | 주 2~3회 |
| 글자 수 | 최소 1,000자 이상 |
| 제목 형식 | `[마케팅 팁] 소상공인이 꼭 알아야 할 ~` |
| 이미지 | 본문 내 최소 3장 이상 |
| 톤 | 정보성, 신뢰감 있는 문체 |

### 카카오톡 채널

| 항목 | 기준 |
|------|------|
| 메시지 발송 | 월 2~4회 (과도한 발송 금지) |
| 내용 | 이벤트, 신규 서비스, 유용한 팁 |
| 응답 시간 | 평일 1시간 이내 |
| 톤 | 가장 친근하게, 짧고 명확하게 |

## 8. 이미지 & 비주얼 스타일

### 사진 스타일 가이드

**✅ 사용 권장 이미지**
- 밝고 자연스러운 조명의 실제 작업 현장
- 사람이 등장하는 따뜻한 분위기의 사진
- 깔끔한 배경의 제품/서비스 연출 컷
- 데이터, 그래프 등 성과를 보여주는 이미지

**❌ 사용 금지 이미지**
- 지나치게 보정되거나 인위적인 스톡 사진
- 어둡고 무거운 분위기의 이미지
- 경쟁사 로고 또는 상표가 보이는 이미지
- 저작권 불명확한 인터넷 다운로드 이미지

### 그래픽 디자인 원칙

| 원칙 | 내용 |
|------|------|
| **여백** | 충분한 여백으로 가독성 확보 (콘텐츠의 30% 이상) |
| **정렬** | 그리드 기반 정렬, 비대칭은 의도적으로만 |
| **폰트 혼용** | 한 디자인에 서체 최대 2종 이내 |
| **컬러** | 주요 컬러 최대 3가지 이내 사용 |
| **아이콘** | 라인 스타일 아이콘 통일 사용 |

### 카드뉴스 / 인포그래픽 형식

```
표지 (1장)  : 제목 + 브랜드 컬러 배경 + 로고
본문 (3~7장): 핵심 정보 1장당 1메시지
마무리 (1장): CTA + 연락처 + 로고
```

## 9. 금지 사항

### 로고 금지 사항

| 금지 항목 | 설명 |
|----------|------|
| 색상 변경 | 승인된 컬러 외 임의 변경 금지 |
| 비율 변형 | 가로/세로 비율 임의 변경 금지 |
| 회전 | 로고 기울이기 또는 회전 금지 |
| 그림자/효과 | 드롭섀도우, 외곽선 등 효과 추가 금지 |
| 배경 충돌 | 가독성 저하 배경에 사용 금지 |
| 부분 사용 | 로고의 일부만 잘라서 사용 금지 |

### 콘텐츠 금지 사항

- 타 브랜드 또는 경쟁사 비방 내용
- 확인되지 않은 수치 또는 과장된 약속
- 저작권 없는 이미지, 음악, 폰트 사용
- 정치적 편향성이 있는 콘텐츠
- 특정 종교, 성별, 연령, 지역 비하 표현
- 개인정보가 포함된 고객 정보 무단 공개
